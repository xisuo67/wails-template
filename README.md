## 如何运行

1、Build Environment

- Node.js [link](https://nodejs.org/en)
- pnpm ：`npm i -g pnpm`
- Go [link](https://go.dev/)
- Wails [link](https://wails.io/) ：`go install github.com/wailsapp/wails/v2/cmd/wails@latest`

2、Pull and Build

```bash
git clone https://github.com/xisuo67/wails-template.git

cd wails-template

wails build
```
